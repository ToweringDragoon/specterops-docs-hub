Full OpenGraph support requires a PostgreSQL graph database and one of the following editions:

- BloodHound Enterprise (uses PostgreSQL by default)
- BloodHound Community v8.0.0+ (requires  changing to a [PostgreSQL database](https://github.com/SpecterOps/bloodhound-docs/blob/main//get-started/custom-installation#postgresql))

  <Note>
    While many OpenGraph features may work on a Neo4j database, there are functional and performance limitations (see the [OpenGraph FAQ](https://github.com/SpecterOps/bloodhound-docs/blob/main//opengraph/faq#why-is-it-taking-so-long-to-ingest-opengraph-data)). For full support, migrate to a PostgreSQL database.
  </Note>
