After you set the required configuration parameters, [run](https://github.com/SpecterOps/bloodhound-docs/blob/main//openhound/community#collect) OpenHound to start the {source} collector and collect data from your {source} {env}. The collector will generate JSON files in the output directory that can be uploaded to BloodHound for analysis.
