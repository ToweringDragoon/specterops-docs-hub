## General Information

The traversable jamf_Update_Self_and_Update_Roles edge represents an API client that possesses 'Update API Integrations' and 'Update API Roles' permissions and at least one role exists. This allows the client to update itself or other API clients to assign any permissions by modifying existing roles. Traversable because the source is already an authenticated API client.
