# API Endpoints

|        |                                    |                                                                                      |
| ------ | ---------------------------------- | ------------------------------------------------------------------------------------ |
| `POST`   | [/api/v2/custom-nodes](https://github.com/SpecterOps/bloodhound-docs/blob/main//reference/custom-node-management/create-custom-nodes)               | Create a new custom node type with configuration.                                    |
| `GET`    | [/api/v2/custom-nodes](https://github.com/SpecterOps/bloodhound-docs/blob/main//reference/custom-node-management/get-custom-nodes)               | Get all custom node types and their configurations.                                  |
| `GET`    | [/api/v2/custom-nodes/\{kind_name\}](https://github.com/SpecterOps/bloodhound-docs/blob/main//reference/custom-node-management/get-custom-node) | Get configuration for a specific node type.                                          |
| `PUT`    | [/api/v2/custom-nodes/\{kind_name\}](https://github.com/SpecterOps/bloodhound-docs/blob/main//reference/custom-node-management/update-custom-node) | Update the configuration of an existing node type.                                   |
| `DELETE` | [/api/v2/custom-nodes/\{kind_name\}](https://github.com/SpecterOps/bloodhound-docs/blob/main//reference/custom-node-management/delete-custom-node) | Delete a custom node type.                                                           |
| `POST`   | [/api/v2/clear-database](https://github.com/SpecterOps/bloodhound-docs/blob/main//reference/database/delete-your-bloodhound-data)             | Delete selected BloodHound data, including targeted graph data such as HasSession edges. |
| `GET`    | [/api/v2/extensions](https://github.com/SpecterOps/bloodhound-docs/blob/main//reference/opengraph-experimental/list-opengraph-extensions-information) | Get a list of all OpenGraph extensions.                      |
| `PUT`    | [/api/v2/extensions](https://github.com/SpecterOps/bloodhound-docs/blob/main//reference/opengraph-experimental/upserts-the-opengraph-extension) | Upserts the OpenGraph extension.                      |
| `DELETE` | [/api/v2/extensions/\{extension_id\}](https://github.com/SpecterOps/bloodhound-docs/blob/main//reference/opengraph-experimental/delete-opengraph-extension) | Delete an OpenGraph extension.                      |
| `GET`    | [/api/v2/extensions-edges](https://github.com/SpecterOps/bloodhound-docs/blob/main//reference/opengraph-experimental/list-edge-kinds) | Get a list of all edge kinds across OpenGraph schemas.                      |
