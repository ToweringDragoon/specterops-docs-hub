## Abuse Info

This edge is not abuseable by itself. When combined with [GetChanges](https://github.com/SpecterOps/bloodhound-docs/blob/main//resources/edges/get-changes), BloodHound will create the abuseable edge [DCSync](https://github.com/SpecterOps/bloodhound-docs/blob/main//resources/edges/dc-sync).

## Opsec Considerations

This edge has no opsec considerations.
