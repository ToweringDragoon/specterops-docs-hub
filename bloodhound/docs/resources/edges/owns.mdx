## Abuse Info

With ownership of the object, you may modify the DACL of the object however you wish. For more information about that, see the [WriteDacl](https://github.com/SpecterOps/bloodhound-docs/blob/main//resources/edges/write-dacl) edge section.

## Opsec Considerations

This depends on the target object and how to take advantage of this privilege.
