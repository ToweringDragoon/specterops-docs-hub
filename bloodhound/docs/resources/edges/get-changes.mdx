## Abuse Info

This edge is not abuseable by itself.

When combined with [GetChangesAll](https://github.com/SpecterOps/bloodhound-docs/blob/main//resources/edges/get-changes-all), BloodHound will create the abuseable edge [DCSync](https://github.com/SpecterOps/bloodhound-docs/blob/main//resources/edges/dc-sync).

When combined with [GetChangesInFilteredSet](https://github.com/SpecterOps/bloodhound-docs/blob/main//resources/edges/get-changes-in-filtered-set), BloodHound will create the abuseable edge [SyncLAPSPassword](https://github.com/SpecterOps/bloodhound-docs/blob/main//resources/edges/sync-laps-password).

## Opsec Considerations

This edge has no opsec considerations.
