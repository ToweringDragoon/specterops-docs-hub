Labels let you tag groups of objects for easier searching and filtering. Use labels to mark objects with common characteristics (e.g., "PCI", "Owned", "Production").

You can filter views and queries based on labels. For example, label PCI-scoped systems to quickly identify attack paths from non-PCI to PCI environments.

### Create a label

Creating a label involves configuring the label details and defining a selector.

<Steps>
  <Step title="Open the Privilege Zones page">
  In the left menu, click **Privilege Zones** > **Labels** > **Create Label**.
  </Step>

  <Step title="Configure initial label details">
  Enter all relevant information about the label:

  | Field                | Required?    | Description                                                                                   |
  |----------------------|:------------:|-----------------------------------------------------------------------------------------------|
  | Name                 | Yes          | A unique name for the label (e.g., PCI)                                                       |
  | Description          | No           | A brief description of the label's purpose and scope (e.g., PCI assets)                       |

    <img
      src="/images/privzones/create-label.png"
      alt="Configure a new privilege zone label"
      style={{ width:"70%" }}
    />

  </Step>

  <Step title="Define a selector">
  Click **Define Selector** to save your new label and continue on to define the objects to include in the label.

  <SelectorWarning/>

  When defining a selector during the label creation process, provide the following information:

  | Field                  | Required?          | Description                                                                                   |
  |------------------------|:------------------:|-----------------------------------------------------------------------------------------------|
  | Name                   | Yes                | A unique name for the selector (e.g., PCI)                                                    |
  | Description            | No                 | A brief description of the selector's purpose and scope (e.g., PCI assets)                    |
  | Selector Type          | Yes                | The type of selector to use (e.g., Object ID or Cypher)                                       |

    <img
      src="/images/privzones/define-label-selector.png"
      alt="Define a selector for the privilege zone label"
      style={{ width:"100%" }}
    />
  </Step>
    <Step title="Complete label creation">
    Click **Save** to finish creating the label.
    </Step>
</Steps>

### Edit or delete a label

To edit or delete an existing label, follow these steps:

<Steps>
  <Step title="Select a label">
   Navigate to the **Privilege Zones** page, select the label you want to edit or delete, and click **Edit**.
  </Step>

  <Step title="Edit or delete a label">
    Choose one of the following actions:

    <Tabs>
      <Tab title="Edit a label" icon="edit">
      To edit the label:

      1. Modify the label's name and description.

      1. Click **Save Edits** to apply your changes.
      </Tab>

      <Tab title="Delete a label" icon="trash-can">
      To delete the label:

      1. Click <Icon icon="trash-can" /> **Delete Label** at the bottom of the page
      1. Confirm your action in the dialog.

          <img
            src="/images/privzones/confirm-label-delete.png"
            alt="Confirm deletion of a custom privilege zone"
            style={{ width:"70%" }}
           />

      1. Click **Confirm** to delete the label.
      </Tab>
    </Tabs>
  </Step>
</Steps>
