Learn how to modify Tier Zero membership in BloodHound Enterprise.

## Purpose

Administrators should use it whenever a need to manually update Tier Zero arises.

## Prerequisites

This process requires an Administrator account in BloodHound Enterprise.

## Process


1.  Open the **Group Management** page
    <Frame>
      <img src="/images/tier_zero/28842139193883.png"/>
    </Frame>
2.  In the left-hand side, the "Add or Remove Members" input box allows you to [search](https://github.com/SpecterOps/bloodhound-docs/blob/main//analyze-data/bloodhound-gui/explore-objects) and select the objects you'd like to add to or remove from Tier Zero.
    <Frame>
      <img src="/images/tier_zero/28842168428699.png"/>
    </Frame>
3.  Once all changes are queued up, click on **Confirm Changes**.
    <Frame>
      <img src="/images/tier_zero/28842139211035.png"/>
    </Frame>

## Outcome

Once Tier Zero has changed, the BloodHound Enterprise tenant will start a reanalysis of Attack Paths, once completed the Attack Paths page will be updated accordingly.
