# Mythic

Mythic is a cross-platform Command & Control framework for red teaming operations.

## Key Features

- **Cross-Platform**: Supports Windows, macOS, and Linux
- **Web-based Interface**: Modern web UI for operators
- **Extensible**: Plugin-based architecture
- **Multi-User**: Collaborative operations support
