# SharpSCCM

SharpSCCM is a post-exploitation tool designed to leverage Microsoft SCCM for lateral movement and credential gathering.

## Key Features

- **Credential Gathering**: Extract Network Access Account credentials
- **Lateral Movement**: Leverage SCCM for movement across the network
- **NTLM Coercion**: Force authentication from SCCM clients and servers
- **CMPivot Integration**: Execute queries via AdminService API
