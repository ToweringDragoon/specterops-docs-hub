# BloodHound

BloodHound uses graph theory to reveal hidden relationships and attack paths in Active Directory environments.

## Key Features

- **Attack Path Discovery**: Identifies complex attack paths automatically
- **Graph Visualization**: Interactive graph interface for exploring data
- **Cross-Platform**: Works on Windows, macOS, and Linux
- **Active Directory & Azure**: Supports both on-premises and cloud environments
